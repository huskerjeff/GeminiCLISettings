# Test Skill

This skill is a test to verify that the Gemini CLI can correctly discover, load, and use custom skills in your environment.

## Verification Steps
1. Run `/skills list` to see if `test-skill` appears.
2. If it appears, this confirms the skill is correctly placed in your global `.gemini/skills` directory.
3. To see more info about this test, check [test-info.md](references/test-info.md).
